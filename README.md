# JavaFX Beads Store Application

## Overview
This project is a JavaFX desktop application that simulates an online beads store where users can customize and purchase jewelry items. The application allows users to create different types of jewelry, add them to a cart, and generate a receipt at checkout.

This project was developed to demonstrate object-oriented programming concepts, GUI design using JavaFX, and file handling in Java.

---

## Features
- Create and customize jewelry items (Necklace, Bracelet, Ring)
- Object-oriented design using inheritance and polymorphism
- Shopping cart system
- Receipt generation and saving to a text file
- JavaFX graphical user interface
- Resource handling (images and FXML)

---

## Technologies Used
- Java
- JavaFX
- Maven
- Object-Oriented Programming (OOP)
- FXML
- File I/O
